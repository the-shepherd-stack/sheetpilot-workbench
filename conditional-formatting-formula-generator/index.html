Use case

Conditional formatting formula generator for spreadsheet work.

Write a custom conditional formatting formula that returns TRUE for the cells or rows that should be highlighted, with the right anchors for your applied range.

What this page gives you

  • A custom formula for the conditional formatting rule you described.
  • A plain-English read of which references stay locked and which move row by row.
  • Notes on the applied range, TRUE/FALSE output, and how the rule behaves across Excel and Google Sheets.

When to use it

Use this when a color scale or built-in preset will not do the job because the highlight depends on another column, a date comparison, or business logic. Include the applied range because conditional formatting evaluates the formula from the first cell of that range and shifts references from there.

Do not paste the result into a worksheet cell as a normal formula. A conditional formatting formula lives inside the rule dialog, and the anchors are written for the first cell of the applied range.

Worked example

Tasks are in A2:C100, due dates are in column B, and status is in column C. Highlight the whole row when the due date is before today and the status is not Done.

=AND($B2<TODAY(),$C2<>"Done")

The dollar signs lock columns B and C so every row checks its own due date and status. The row number stays relative, so the rule moves from row 2 to row 3, row 4, and beyond.

Check before you paste

  • The formula should return TRUE or FALSE so the formatting rule knows when to fire.
  • Lock columns with dollar signs when the rule should follow row by row, such as $B2.
  • For Google Sheets, use same-sheet references directly and INDIRECT when the rule must reference another sheet.
  • Open the rule on the first cell of your applied range and confirm the highlight lands where you expect before extending it.
